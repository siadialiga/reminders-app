A production-ready, macOS-inspired desktop application for managing tasks and lists. Built with **Tauri**, **React**, and **Rust**, this app delivers a premium experience with native performance and a focus on sleek aesthetics.

## ✨ Features

- **macOS Design Aesthetics**: Clean, minimalist UI with a premium look and feel.
- **True Black Theme**: Deep black dark mode for OLED screens and high-contrast focus.
- **Smart Views**: Automatically organized tasks (Today, Scheduled, All, Flagged, Completed).
- **Custom List Management**: Create, rename, and organize lists.
- **Drag & Drop**: Intuitive list reordering with visual feedback.
- **Task Details**: Rich task editing with notes, due dates, reminder times, and priority levels.
- **Native Integration**:
  - **System Tray**: Runs in the background with a specialized tray menu.
  - **Auto-start**: Option to launch automatically at login.
  - **Notifications**: Get native system notifications for due tasks.
- **i18n Localization**: Full support for English and Turkish languages.
- **Secure Data**: All tasks and settings are stored locally on your device.

## 🚀 Technologies

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: Rust, Tauri.
- **State Management**: Custom React Context API with persistent local storage.
- **Internationalization**: Lightweight custom i18n system.

## 🛠️ Development & Build

### Prerequisites
- [Node.js](https://nodejs.org/)
- [Rust & Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html)
- [Tauri Prerequisites](https://tauri.app/v1/guides/getting-started/prerequisites)

### Commands
```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build production binary
npm run tauri build
```

## 🔒 Privacy & Safety
This application does not collect any user data. All your reminders and information remain on your local machine.
